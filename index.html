28 ივლისი

დღის მოვლენები

1493 : მოსკოვის დიდი ხანძარი.

1540 : ჰენრი VIII-ის ბრძანებით ჩამოახრჩვეს თომას კრომველი, ღალატის ბრალდებით. ჰენრი იმავე დღეს ჯვარს იწერს მეხუთე ცოლზე - კეთრინ ჰოვარდზე.
1751 : კახეთის მეფე ერეკლე II ამარცხებს ირანის ტახტის პრეტენდენტს აზატ-ხანს ყირბულახის ბრძოლაში.
1794 : საფრანგეთის რევოლუციის 22 მოღვაწეს, მათ შორის მაქსიმილიან რობესპიერსა და ლუი-ანტუან სენ-ჟუსტს საჯაროდ გილიოტინაზე თავი მოკვეთეს.
1821 : პერუმ ესპანეთისაგან დამოუკიდებლობა გამოაცხადა.
1914 : ავსტრია-უნგრეთმა ომი გამოუცხადა სერბიას. დაიწყო პირველი მსოფლიო ომი.
1943 : მეორე მსოფლიო ომი: ოპერაცია გომორა - ბრიტანელები ბომბავენ ჰამბურგს, რის მიერაც გამოწველმა ხანძარმა 42 000 გერმანელი მოქალაქე იმსხვერპლა.
1976 : 8,2 მაგნიტუდის ტანგშანის მიწისძვრამ ჩინეთში 242 769 ადამიანი იმსხვერპლა, 164 851 დაშავდა.
2005 : დროებითი ირლანდიური რესპუბლიკური არმია (ირა) ჩრდილოეთ ირლანდიაში ოცდაათწლიანი შეიარაღებული კამპანიის შეწყვეტის შესახებ აცხადებს.
2017 : პაკისტანის პრემიერ-მინისტრი ნავაზ შარიფი უზენაესმა სასამართლომ პანამის დოკუმენტების მონაცემთა გადამოწმების შემდეგ თანამდებობიდან გადააყენა.
2021 : ქართველი ძიუდოისტი ლაშა ბექაური -90 კგ წონით კატეგორიაში ოლიმპიური ჩემპიონი გახდა.

1812 : კრაშევსკი, იუზეფ იგნაცი — პოლონელი მწერალი, პუბლიცისტი და ისტორიკოსი
1844 : ჰოპკინზი, ჯერარდ მანლი — ინგლისელი პოეტი
1874 : კასირერი, ერნსტ — გერმანელი ფილოსოფოსი
1887 : დიუშანი, მარსელ — ფრანგი მხატვარი (გ.1968)
1902 : პოპერი, კარლ — ფილოსოფოსი (გ.1994)
1904 : ჩერენკოვი, პავლე — რუსი ფიზიკოსი, ნობელის პრემიის ლაურეატი
1927 : გიორგაძე, გრიგოლ — ისტორიკოსი (გ. 2005),
1929 : ჟაკლინ კენედი ონასისი — აშშ-ს პრეზიდენტ კენედი, ჯონ მეუღლე
1938 : ალბერტო ფუხიმორი — პერუს პრეზიდენტი
1941 : მუტი, რიკარდო — იტალიელი დირიჟორი
1951 : კალატრავა, სანტიაგო — ესპანელი არქტიტექტორი
1952 : ვაჯირალონგკორნი — ტაილანდის პრინცი
1961 : ალექსანდრე კურლოვიჩი — ბელარუსი და საბჭოთა ძალოსანი, ორგზის ოლიმპიური ჩემპიონი

1791 : რობესპიერი, მაქსიმილიან — ფრანგი რევოლუციონერი ოღვაწე,
1844 : ჟოზეფ ბონაპარტი — ნაპოლეონის უფროსი ძმა, ნეაპოლის და ესპანეთის მეფე
1968 : ჰანი, ოტო — გერმანელი ფიზიკოსი და რადიო-ქიმიკოსი
1972 : მაჯუმდალი, ჩარუ — ინდოელი რევოლუციონერი
2010 : მზია კვირიკაშვილი — ქართველი მსახიობი, იუმორისტი - საქართველოს სიცილის დედოფალი (დ.1949)
